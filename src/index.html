sc-gallery.js

The sc-gallery.js is an easily configurable plugin for image galleries. Click to show next image adapted to main container.

Demo 1

In this demo case, a gallery with different size images.

Gallery

Code

<html>
<head>
    <link rel="stylesheet" type="text/css" href="sc-gallery.css">
    <script src="sc-gallery.js"></script>
</head>
<body>
<div class="sc-gallery">
    <img src="img/turtle-shadow.jpg" alt="turtle-shadow">
    <img src="img/crimson-sentinel.jpg" alt="crimson-sentinel">
    <img src="img/neon-vanguard.jpg" alt="neon-vanguard">
    <img src="img/raccoon-guardian.jpg" alt="raccoon-guardian">
    <img src="img/cardboard-crusader.jpg" alt="cardboard-crusader">
</div>
</body>
</html>

Demo 2

In this demo case, a gallery with same size images and custom style.

Gallery

Code

<html>
<head>
    <link rel="stylesheet" type="text/css" href="sc-gallery.css">
    <script src="sc-gallery.js"></script>
    <style>
        .sc-gallery.custom-1 {
            width: 250px;
            height: 250px;
            padding: 1rem;
        }

        @media screen and (max-width: 749px) {
            .sc-gallery.custom-1 {
                width: 50vw;
                height: 50vw;
            }
        }
    </style>
</head>
<body>
<div class="sc-gallery custom-1">
    <img src="img/turtle-shadow.jpg" alt="turtle-shadow">
    <img src="img/crimson-sentinel.jpg" alt="crimson-sentinel">
    <img src="img/neon-vanguard.jpg" alt="neon-vanguard">
    <img src="img/raccoon-guardian.jpg" alt="raccoon-guardian">
    <img src="img/cardboard-crusader.jpg" alt="cardboard-crusader">
</div>
</body>
</html>
